# 🏊‍♂️ Corrida das Sílabas

Um jogo educativo digital desenvolvido para auxiliar crianças da **APAE de Andirá** no processo de alfabetização e consciência fonológica utilizando o método **ABACADA**.

O jogo transforma o aprendizado em uma experiência divertida e inclusiva através de uma competição de natação entre dois jogadores.

---

# 📚 Sobre o Projeto

O projeto foi criado com o objetivo de estimular:

* Identificação de sílabas
* Consciência fonológica
* Associação entre imagem e som
* Aprendizado lúdico e inclusivo
* Participação social entre crianças

A proposta pedagógica utiliza elementos visuais, interação simples e feedback positivo para incentivar o aprendizado sem punição ao erro.

---

# 🎮 Como Funciona

Dois jogadores participam de uma corrida de natação dentro de uma piscina.

Cada jogador deve identificar corretamente a sílaba correspondente à palavra exibida na carta.

Quando o jogador acerta:

* O personagem avança uma casa na piscina.

O primeiro jogador que chegar ao final da raia vence a partida.

---

# 🧠 Método ABACADA

O jogo utiliza o método ABACADA para trabalhar sílabas simples de forma visual e intuitiva.

Exemplos:

* BA
* BE
* BI
* BO
* BU

As crianças aprendem identificando sílabas presentes em palavras do cotidiano.

---

# 🕹️ Mecânica do Jogo

## Passo 1 — Carta na tela

Uma carta aparece contendo:

* Imagem
* Palavra relacionada

Exemplo:

* Cachorro
* Bola
* Casa

---

## Passo 2 — Escolha da sílaba

O jogador recebe 3 opções de sílabas.

Apenas uma alternativa está correta.

---

## Passo 3 — Resposta

O jogador seleciona uma das opções disponíveis.

### Feedback visual:

* 🟢 Verde → resposta correta
* 🔴 Vermelho → resposta incorreta

O botão correto sempre aparece destacado para reforçar o aprendizado.

---

## Passo 4 — Movimento

Ao acertar:

* O personagem avança na piscina.

---

# 🏊 Cenário

O jogo se passa em uma piscina com:

* 2 raias
* Casas numeradas
* Linha de chegada
* Personagens animados

O objetivo é alcançar o final da piscina antes do adversário.

---

# 👦 Personagens

O projeto possui personagens infantis inclusivos, representando diversidade e acessibilidade.

Cada jogador escolhe um personagem antes do início da partida.

---

# 🃏 Sistema de Cartas

Atualmente o jogo possui:

* 12 cartas cadastradas
* Palavras infantis simples
* Sílabas compatíveis com o método ABACADA

As cartas:

* São embaralhadas
* Não repetem imediatamente
* Possuem alternativas aleatórias
